Dog Breed Classifier

Using a modified DenseNet121 neural network and OpenCV, this app will predict the breed of a dog in an image and output the top five classes. It will handle cases for a dog, person, and not a dog. If there is a person in the image the model will return the dog breed that the person most resembles. The associated code for training the model using PyTorch can be found on my Github page for this project. Please upload a jpg imge less than 1 MB.
